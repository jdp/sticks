Sticks is up and running.

You can now start developing your application.

To get started, all you need to do is edit /.html (this file) and /.php. It might also be a good idea to read the documentation, if you haven't done that already.

Quick Reference

  • $sticks->template is the name of the template you wish to use for a script. It defaults to default.
  • $sticks->script_output is the output generated by a script. Put it in your template files where you want the script output to show up.
  • $sticks->url($target) will generate a correct Sticks link. The $target argument is a string with the format script/arg1/arg2/...
